_Ascii3dxmlParser__find_in_list(seq, name, index=False)¶

Searches a element with given name in a sequence. This function will work e.g. to search meshes in a scene_pb2.Library instance, a graphical mesh in a scene_pb2.GraphicalScene instance... + param: seq is the sequence in which the element is searched for. + param: name is the name of the element to locate. + param: index indicates whether to return the index of the element in the list. + return: the element if it was found, None otherwise.
