agents.physic.builder.deserializePhysicalScene(phy, gvm_scene, xcd_scene, physical_scene)¶

This function deserializes the physical part of a world. It does not deserialize the collision scene, collision pairs and association between collision scene and physical scene. Also note that deserializing the collision/physics bingings must be done prior to deserialization of collision pairs: 1/ deserialize collision 2/ deserialize physics 3/ deserialize bindings 4/ deserialize collision pairs If you deserialize these elements in one block, a shortcut is to use the deserializeWorld function provided in this module.
