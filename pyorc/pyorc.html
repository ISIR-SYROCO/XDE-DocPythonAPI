class pyorc.Task(name, model, controller, frame_name, kind, **args)¶

Bases: object

Builds and wraps a C++ orc::Task.

Creates a task. frame_name is either ‘CoM’ or the name of a segment of the controlled manikin (e.g. ‘Right Center of Prehension’). A local frame can be specified using the optional argument Hloc. The kind of task can be specified using the argument kind. Possible values are ‘position’, ‘orientation’, ‘displacement’. For position and displacement tasks, axes can be specified using the optional argument axes. The optional arguments can also be used to specified a bandwidth, the gains, and the weight of the task (for controllers that handle priorities). Note that if you specify both a bandwith and a stiffness or damping, the stiffness and damping that you specify will overwrite the value computed using the specified bandwidth.

setTargetAltitude(z, **args)¶

Modify the target of a task to specify a desired altitude. The target is either specified in arguments using fromTarget=, or taken from the current state of the manikin; then the altitude of the target is overwritten with the value z specified in argument.

setTargetLookAt(v_local, vref_abs, **args)¶

This function can be used if you want to implement a ‘look at’ behavior using an orientation task. v_local is a vector specified in the local frame (the control frame associated with the task), vref_abs is the desired coordinates of this vector in world coordinates. The target is computed either from the current state or from a specified displacement using fromTarget=.

startTask(**args)¶

Starts a task. It is possible to force the desired state to match the current state by setting resetDesiredState=True in the arguments list.
